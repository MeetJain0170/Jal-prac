System Overview

JalDrishti is a maritime AI surveillance platform combining deep learning image restoration with real-time object detection and environmental intelligence.

U-Net CNN with multi-scale PINN loss restores visibility. YOLOv8-World detects and classifies 80+ maritime object classes. MiDaS provides monocular depth estimation. Dark Channel Prior analysis quantifies turbidity and water type.
