# Challenges and Limitations

## Problems Faced During Development
- Missing or unavailable model files during setup.
- SSL/network issues while downloading external model dependencies.
- False positives in underwater detection under varied scene types.
- Balancing enhancement strength without color artifacts.

## Limitations of the System
- Domain shift across different underwater environments.
- Performance still depends on input quality and visibility.
- Runtime behavior may vary slightly across systems due to dependency versions.
- Some third-party weights may download on first run if cache is absent.
